# Space Striker 🚀

A retro 2D pixelated arcade space shooter game built using HTML5, CSS, and JavaScript.

## 🎮 How to Play
* **Movement:** Drag your finger on the screen, or drag with your mouse.
* **Keyboard:** Use `W`, `A`, `S`, `D` or the `Arrow Keys`.
* **Shooting:** The ship auto-fires continuously!
* **Power-ups:** Collect Yellow (Multi-shot), Cyan (Shield), Orange (Speed), and Pink (Laser) gems!
