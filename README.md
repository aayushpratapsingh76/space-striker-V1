# Space Striker 🚀

A retro 2D pixelated arcade space shooter game built using HTML5, CSS, and JavaScript. This project is a small web-based prototype designed to showcase smooth mobile/desktop controls and classic arcade progression.

## 🕹️ Live Prototype
Play the game instantly in your browser here: 
👉 **[https://space-striker-v1.vercel.app/](https://space-striker-v1.vercel.app/)**

---

## 🎮 Key Features
* **Core Gameplay:** Fast-paced arcade action with smooth scrolling backgrounds and auto-firing continuous shooting physics.
* **3 Epic Boss Battles:** Survive increasingly difficult waves to trigger unique boss encounters, complete with dedicated movement patterns and health bars.
* **Dynamic Power-up Gems:** Upgrade your ship mid-fight by collecting specialized power-up gems drop-tested for strategic variety.

---

## 🕹️ How to Play

### Controls
* **Touch/Mobile:** Drag your finger anywhere on the screen to smoothly navigate the ship.
* **Desktop/Mouse:** Click and drag with your mouse pointer.
* **Keyboard Fallback:** Use `W`, `A`, `S`, `D` or the standard `Arrow Keys`.

### Power-up Guide
Catch falling gems to instantly activate tactical advantages:
* 🟨 **Yellow:** Multi-shot weapon upgrade
* 🟦 **Cyan:** Active energy shield protection
* 🟧 **Orange:** Engine propulsion speed boost
* 🟥 **Pink:** High-damage continuous laser beam
